# CUDA-Q Example Plugins

This directory contains example external plugins that demonstrate how to
extend CUDA-Q without modifying the core source tree. Each plugin is a
standalone external project with its own documentation, build instructions,
and tests.

| Plugin | Shape | What it demonstrates |
|--------|-------|---------------------|
| [`mock_rest/`](mock_rest/README.md) | REST | `ServerHelper` subclass for the built-in `remote_rest` QPU |

See each plugin's README for its design, prerequisites, build and test commands,
installation steps, and runnable examples.
