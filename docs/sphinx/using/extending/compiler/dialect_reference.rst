CUDA-Q dialect documentation
****************************

These pages document the CUDA-Q dialects. Each page includes content generated
from the dialect's TableGen definitions, including its operations, types,
assembly syntax, traits, and constraints. For how the dialects are used together
in compiler IR, see :doc:`cudaq_ir`.
